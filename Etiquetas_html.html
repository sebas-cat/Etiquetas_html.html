    ol

    Esta etiqueta permite colocar viñetas a un elemento, puede ser con numeración o alfabeticamente

    Viñetas numéricas

    1. ejemplo1
    2. ejemplo2
    3. ejemplo3

    Viñetas alfabéticas

    1. ejemplo1
    2. ejemplo2
    3. ejemplo3

    option

    Se utiliza normalmente para definir los elementos a escoger de un menú desplegable

    optgroup

    Permite agrupar los elementos "option", permitiendo organizarlos por categorias en el menú desplegable

    output

    Esta etiqueta permite desplegar el resultado de un operación realizada por el ususario

    + = 3

    p

    Esta etiqueta permite colocar párrafos o bloques de texto

    --Esto es un ejemplo de un párrafo--

    param

    Esta etiqueta se encuentra obsoleta y en muchos navegadores no es admitida, pero anteriormente era utilizada para otorgar parámetros a un elemento "object", es decir, indica al navegador cómo debe comportarse ese elemento

    pre

    Permite desplegar un bloque de texto preformateado, lo que implica que se va a presentar como tal esté escrito. 

             *
        ***
       *****
      *******

    progress

    La etiqueta progress se utiliza para representar el progreso de una tarea en curso, mostrando qué parte ya está completada.

    40%

    q

    La etiqueta q se utiliza para representar citas cortas dentro del texto.

    Einstein dijo: La imaginación es más importante que el conocimiento.

    ruby

    La etiqueta ruby se utiliza para mostrar anotaciones sobre caracteres, muy común en idiomas asiáticos. Funciona junto con rt y rp.

    漢 Kan 字 Ji (Caracter chino)

    rt

    Permite definir una explicación o pronunciación en una anotación ruby

    漢 ㄏㄢˋ

    rp

    Permite proporcionar un paréntesis alrededor de un texto ruby, ayuda a definir qué mostrará el navegador si no admite anotaciones ruby

    漢 (ㄏㄢˋ)

    samp

    La etiqueta samp se utiliza para mostrar ejemplos de salida de un programa o sistema.

    Error 404: Página no encontrada
